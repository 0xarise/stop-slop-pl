# Struktury do unikania

## Kontrast binarny

Tworzą fałszywy dramat. Stwierdź wprost.

| Wzorzec | Problem |
|---------|---------|
| "To nie jest X. To Y." / "Nie chodzi o X, ale o Y." | Telegrafowane odwrócenie |
| "Problem nie leży w X, lecz w Y." | Formulaiczne przeramowanie |
| "Odpowiedź to nie X. To Y." | Przewidywalny pivot |
| "Wygląda jak X. Tak naprawdę to Y." | Schemat setup/reveal |
| "nie X, ale Y" / "nie X, a Y" | Mechaniczny kontrast |
| "przestaje być X i zaczyna być Y" | Fałszywy łuk transformacji |
| "nie tylko X, ale również Y" | Addytywna zagrywka |

**Zamiast tego:** Stwierdź Y wprost. "Problemem jest Y." Wyrzuć negację.

## Negatywne listowanie

Listowanie czym coś NIE jest przed ujawnieniem czym JEST.

| Wzorzec | Problem |
|---------|---------|
| "To nie jest szkolenie. To nie jest prezentacja. To jest..." | Buildup przez negację |
| "To nie było X. To nie było Y. To było Z." | Ten sam schemat, czas przeszły |

**Zamiast tego:** Stwierdź Z. Czytelnik nie potrzebuje rozbiegu.

## Dramatyczna fragmentacja

Urywane zdania dla emfazy brzmią jak produkowana głębia.

| Wzorzec | Problem |
|---------|---------|
| "Jedno słowo: [słowo]." | Performatywna prostota |
| "X. I Y. I Z." | Staccato drama |
| "X. I tyle." / "X. Kropka." | Dramatyczna finalizacja |

**Zamiast tego:** Pełne zdania. Ufaj treści, nie prezentacji.

## Retoryczne ustawianie

Zapowiadają wgląd zamiast go dostarczać.

| Wzorzec | Problem |
|---------|---------|
| "A co gdyby...?" / "Wyobraź sobie, że..." | Sokratejska poza |
| "Oto co mam na myśli:" | Redundantny podgląd |
| "Pomyśl o tym:" / "Zastanów się:" | Protekcjonalne naprowadzanie |
| "I to jest w porządku." | Zbędne pozwolenie |
| "Pozwolę sobie zauważyć, że..." | Fałszywa skromność |

**Zamiast tego:** Przejdź do meritum. Czytelnik sam wyciągnie wnioski.

## Fałszywa sprawczość

Dawanie nieożywionym obiektom ludzkich czasowników.

| Wzorzec | Problem |
|---------|---------|
| "rynek wymaga" | Kupujący oczekują. |
| "technologia umożliwia" | Używając [narzędzia], możesz [co]. |
| "dane pokazują" | Ktoś czyta dane i wyciąga wniosek. |
| "branża się transformuje" | Firmy zmieniają zachowanie. |
| "decyzja wyłania się" | Ktoś decyduje. |
| "rozmowa zmierza ku" | Ktoś ją prowadzi. |

**Zamiast tego:** Nazwij człowieka. "Zespół naprawił to w tydzień" wygrywa z "problem został rozwiązany."

## Narrator z dystansu

Unoszenie się nad sceną zamiast postawienia czytelnika w niej.

| Wzorzec | Problem |
|---------|---------|
| "Nikt tego nie planował." | Bezcielesna obserwacja |
| "To dzieje się dlatego, że..." | Głos wykładowcy |
| "Ludzie mają tendencję do..." | Socjolog z fotela |
| "Obserwujemy, że..." | Fałszywy naukowy dystans |

**Zamiast tego:** Postaw czytelnika w sytuacji. "Ty" wygrywa z "Ludzie".

## Strona bierna

Każde zdanie potrzebuje podmiotu, który coś robi.

| Wzorzec | Fix |
|---------|-----|
| "Zostało to wdrożone" | Nazwij kto wdrożył |
| "Uważa się, że" | Nazwij kto uważa |
| "Podjęta została decyzja" | Nazwij kto zdecydował |
| "Przygotowany został raport" | Nazwij kto przygotował |

## Formulaiczne otwarcia akapitów

| Wzorzec | Fix |
|---------|-----|
| "Warto wspomnieć, że..." | Usuń, przejdź do treści |
| "Nie bez znaczenia jest fakt, że..." | Stwierdź znaczenie wprost |
| "Z perspektywy [X]..." | Zwykle filler, sprawdź czy konieczne |
| "Należy podkreślić, że..." | Jeśli ważne, czytelnik zrozumie |
| "Nie można nie zauważyć, że..." | Podwójna negacja, stwierdź wprost |

## Wzorce rytmu

| Wzorzec | Fix |
|---------|-----|
| Trzy-elementowe listy | Dwa elementy lub jeden |
| Każdy akapit kończy się puentą | Zróżnicuj zakończenia |
| Zdania o tej samej długości pod rząd | Złam jedno |
| Myślniki jako gadżet stylistyczny | Kropki i przecinki |
| Piętrzenie krótkich zdań-ciosów | Mieszaj długości |

## Wzorce słowne

| Wzorzec | Problem |
|---------|---------|
| Leniwe ekstrema (każdy, zawsze, nigdy, wszyscy, nikt) | Fałszywy autorytet. Użyj konkretu. |
| Przysłówki-wypełniacze (niezwykle, zdecydowanie, niewątpliwie) | Pusta emfaza. Patrz phrases.md. |
| Anglicyzmy-buzzwordy (mindset, roadmapa, stakeholderzy) | Użyj polskiego odpowiednika. |
